## EasyShift Monorepo

Это корень проекта EasyShift — платформы для подработок на ПВЗ (бот + мини‑приложение + сайт + мобильное приложение).

Ключевые каталоги:

- `apps` — исполняемые приложения (API, Telegram‑бот, мини‑приложение, веб, мобильное).
- `packages` — общие библиотеки и shared‑код.
- `deploy` — вспомогательные файлы для выкладки на сервер (runtime package и т.п.).
- `docs` — документация и контекст разработки (архитектура, деплой, стратегия БД).
- `node_modules` — зависимости npm (не коммитятся).

Подробный контекст по отдельным подсистемам лежит в `docs/context`.
