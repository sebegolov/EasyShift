## apps

Каталог с прикладными частями системы EasyShift:

- `api` — backend на NestJS + Prisma + PostgreSQL (REST API для всего проекта).
- `telegram-bot` — бот на Telegraf, общается с API и открывает мини‑приложение.
- `mini-app` — Telegram Mini App (React/Vite сборка, в проде используется готовый `dist`).
- `web` — веб‑интерфейс (в том числе будущий бэк‑офис).
- `mobile` — заготовка для мобильного клиента.

Сейчас в репозитории в основном лежат собранные `dist`‑версии, исходники TypeScript/React могут храниться отдельно.
