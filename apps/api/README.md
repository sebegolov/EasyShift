## apps/api — Backend API

NestJS + Fastify API для всего проекта EasyShift.

Что важно:

- `src` — исходники модулярного монолита (пользователи, смены, подписки, админ‑часть и т.д.).
- `dist` — собранная версия, которая крутится на прод‑сервере (`apps/api/dist/main.js`).
- `prisma` — схема базы данных (`schema.prisma`) и миграции (если будут добавлены).

Ключевые решения по данным и аккаунтам описаны в `docs/context/data-storage-strategy.md`.
